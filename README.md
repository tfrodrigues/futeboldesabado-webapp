# FUTEBOL DE SÁBADO - GERENCIAMENTO DE TIMES

Site para gerenciamento de agenda e informações gerais de times amadores

## Instruções para rodar
- Utilizar a versão 0.12.4 do Node. 
- Utilizar a versão 3.0.2 do MongoDB.
- Navegar até o diretório raiz da aplicação.
- Executar o comando ```npm install```.
- Rodar o mongodb com ```mongod --dbpath XXX```, onde XXX é o diretório de armazenamento dos dados.  
- Rodar o node com ```node server.js```
- Acessar a aplicação em http://localhost:4000.
